TP2 — Exercice2

Dans cet exercice, il fallait reproduire le même design de page web de trois manières : d’abord avec CSS Grid sans zones nommées, ensuite avec Grid en utilisant les zones (grid-area), et enfin en utilisant seulement Flexbox. Pour les deux premières parties, il fallait respecter le HTML fourni et tout faire uniquement en CSS, ce qui oblige à bien comprendre comment Grid place les éléments avec grid-template-columns, grid-template-rows et le positionnement (ligne/colonne). J’ai trouvé l’exercice intéressant parce qu’on voit clairement les avantages de Grid pour faire des mises en page “en blocs”, et aussi les limites quand on essaye de refaire la même chose uniquement avec Flexbox. Au début j’ai eu du mal avec le placement exact des items en Grid, surtout quand il faut gérer les tailles en % et en px en même temps. J’ai aussi un peu galéré avec grid-area parce qu’il faut bien visualiser la grille et ne pas se tromper dans les zones.

Connaissances mises en jeu : CSS Grid (lignes/colonnes, grid-template-*, gap, placement), zones nommées (grid-template-areas, grid-area), Flexbox (alignement, répartition), box-model (margin/padding), lecture d’un layout et reproduction fidèle.
